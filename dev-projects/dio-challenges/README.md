\# Desafios de Lógica de Programação - DIO 🚀



Este repositório contém a resolução de três desafios de lógica de programação e pensamento computacional, desenvolvidos em \*\*JavaScript\*\*. Os projetos progridem em complexidade, passando por conceitos fundamentais até a introdução à Programação Orientada a Objetos (POO).



\## 📝 Resumo dos Desafios



\* \*\*1️⃣ Classificador de Nível de Herói (`challenge-1.js`):\*\* Um script focado em fundamentos básicos. Ele utiliza variáveis, operadores lógicos e estruturas de controle de fluxo (`if/else`) para classificar a patente de um herói (como Ferro, Bronze, Ouro, etc.) com base na sua quantidade total de Experiência (XP).



\* \*\*2️⃣ Calculadora de Partidas Rankeadas (`challenge-2.js`):\*\* Este desafio avança para o uso de blocos reutilizáveis de código. Ele implementa uma função que calcula o saldo entre vitórias e derrotas de jogadores. Além disso, utiliza Arrays, Objetos e laços de repetição (`for...of`) para iterar sobre múltiplos jogadores e determinar seus níveis no ranking.



\* \*\*3️⃣ Classes de um Jogo (`challenge-3.js`):\*\* Uma aplicação prática de Programação Orientada a Objetos (POO). O código estrutura uma classe `Heroi` que recebe propriedades através de um método construtor (nome, idade e tipo). Ele também possui um método `atacar()` que, através de uma estrutura `switch/case`, identifica o tipo do herói (mago, guerreiro, monge ou ninja) e exibe o ataque específico correspondente.



\## 💻 Tecnologias Utilizadas



\* \*\*JavaScript:\*\* Linguagem principal utilizada para resolver a lógica dos desafios.

\* \*\*Node.js:\*\* Ambiente de execução recomendado para testar os scripts no terminal.



\## 🛠️ Como executar



Para testar os códigos localmente, certifique-se de ter o \[Node.js](https://nodejs.org/) instalado em sua máquina. Clone este repositório, abra o terminal na pasta dos arquivos e execute o comando correspondente ao desafio que deseja testar:



```bash

node challenge-1.js

node challenge-2.js

node challenge-3.js
