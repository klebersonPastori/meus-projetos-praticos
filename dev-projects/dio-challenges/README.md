Este repositório contém a resolução de três desafios de lógica de programação e pensamento computacional propostos pela **DIO (Digital Innovation One)**, desenvolvidos inteiramente em **JavaScript**. 

Os projetos progridem em complexidade, passando por conceitos fundamentais de controle de fluxo até a introdução prática à Programação Orientada a Objetos (POO).

## 📝 Resumo dos Desafios

### 1️⃣ Classificador de Nível de Herói (`challenge-1.js`)
Focado em fundamentos básicos de sintaxe. Utiliza variáveis, operadores lógicos e estruturas de controle de fluxo (`if/else`) para classificar a patente de um herói (como Ferro, Bronze, Ouro, etc.) com base na sua quantidade total de Experiência (XP).

### 2️⃣ Calculadora de Partidas Rankeadas (`challenge-2.js`)
Avança para a criação e uso de funções (blocos reutilizáveis de código). Implementa uma lógica que calcula o saldo entre vitórias e derrotas. Além disso, utiliza Arrays, Objetos e laços de repetição (`for...of`) para iterar sobre múltiplos jogadores e determinar seus níveis no ranking.

### 3️⃣ Classes de um Jogo (`challenge-3.js`)
Uma aplicação prática de **Programação Orientada a Objetos (POO)**. O código estrutura uma classe `Heroi` que recebe propriedades através de um método construtor (nome, idade e tipo). Possui também um método `atacar()` que, através de uma estrutura `switch/case`, identifica o tipo do herói (mago, guerreiro, monge ou ninja) e exibe o ataque específico correspondente.

## 💻 Tecnologias Utilizadas

* **JavaScript:** Linguagem principal utilizada para construir a lógica dos desafios.
* **Node.js:** Ambiente de execução utilizado para testar os scripts localmente no terminal.

## 🛠️ Como Executar Localmente

Para testar os códigos na sua máquina, certifique-se de ter o [Node.js](https://nodejs.org/) instalado. 

**1. Clone este repositório:**
```bash
git clone [https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git](https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git)

2. Navegue até a pasta do projeto:

Bash
cd SEU-REPOSITORIO
3. Execute o comando correspondente ao desafio que deseja testar:

Bash
node challenge-1.js
node challenge-2.js
node challenge-3.js
