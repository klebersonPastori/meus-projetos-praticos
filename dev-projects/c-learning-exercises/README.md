# 🧮 Operações Aritméticas em C (Exercício de Aprendizado)

> Autor: **Kleberson Pastori**  
> Linguagem: **C**  
> Propósito: Exercício introdutório de entrada/saída e operadores aritméticos em C.

Este programa lê **dois números inteiros** do teclado e calcula **soma, subtração, multiplicação e divisão** (divisão inteira) entre eles, exibindo os resultados no console. [1](https://hyundaibrasil-my.sharepoint.com/personal/hae_kleberson_pastori_hyundai-brasil_com/Documents/Microsoft%20Copilot%20Chat%20Files/c-learning-exercises.c)

## 📄 O que o código faz

- Declara duas variáveis de entrada: `numero1` e `numero2` (inteiros).  
- Calcula `soma`, `subtracao`, `multiplicacao` e `divisao` usando operadores aritméticos básicos.  
- Imprime cada resultado em uma linha.  
- **Observação:** a divisão é **inteira** (trunca o resultado) e não há verificação de **divisão por zero** no estado atual. [1](https://hyundaibrasil-my.sharepoint.com/personal/hae_kleberson_pastori_hyundai-brasil_com/Documents/Microsoft%20Copilot%20Chat%20Files/c-learning-exercises.c)
