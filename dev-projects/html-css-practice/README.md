# Práticas de Fundamentos: HTML5 e CSS3 🌐

Este repositório contém uma coleção de arquivos desenvolvidos para fixar conceitos fundamentais de desenvolvimento web, focando em estruturação com **HTML5** e estilização básica com **CSS3**. Os projetos englobam desde a criação de páginas estáticas simples até formulários com múltiplos tipos de entradas de dados.

## 📝 Resumo dos Arquivos

* **🏠 Página Inicial (`home.html`):** Uma página de boas-vindas simples que utiliza CSS interno para centralização de conteúdo com `Flexbox`. Ela demonstra o uso de tabelas HTML, estilização de textos (cores no título) e incorporação de mídias externas (GIF).

* **📖 Guia de Estudos HTML (`hmtl-practice.html`):** Uma página estruturada como um mini-tutorial ou glossário (com referências ao módulo da DIO). Explora o uso de âncoras para navegação interna (Índice), formatação de texto (`<i>`, `<strong>`), listas ordenadas (`<ol>`) e não ordenadas (`<ul>`), além de links externos.

* **✉️ Formulário de Contato Estilizado (`forms.html`):** Um formulário de contato com um visual agradável criado através de CSS interno. O arquivo foca em estilização de caixas (`box-shadow`, `border-radius`), efeitos de transição (`hover` no botão) e uso de inputs obrigatórios (`required`) para texto, email, telefone e data.

* **📋 Cadastro de Currículo Avançado (`forms-2.html`):** Um formulário abrangente que explora a grande variedade de inputs oferecidos pelo HTML5. Ele inclui campos como: senhas (`password`), caixas de seleção (`select`), seletores de cor (`color`), barras de intensidade (`range`), botões de rádio (`radio`), caixas de marcação (`checkbox`) e envio de arquivos (`file`).

## 💻 Tecnologias Utilizadas

* **HTML5:** Utilizado para a semântica e estruturação de todo o conteúdo das páginas e formulários.
* **CSS3:** Aplicado internamente (na tag `<style>`) para criar layouts centralizados, aplicar cores, bordas e efeitos visuais simples.

## 🛠️ Como visualizar os projetos

Como são arquivos estáticos focados no front-end (sem necessidade de um servidor back-end local), a visualização é muito simples:

1. Clone este repositório em sua máquina local.
2. Navegue até a pasta onde os arquivos foram salvos.
3. Dê um duplo clique em qualquer um dos arquivos `.html` (ou clique com o botão direito e selecione "Abrir com") e escolha o seu navegador de internet preferido (Chrome, Edge, Firefox, etc.).
