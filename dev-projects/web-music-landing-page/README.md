\# 🎸 Pearl Jam "Ten" - Tribute \& Web Dev Lab 💿



Este projeto é uma página web interativa e responsiva criada como um laboratório prático para o aprimoramento de habilidades em Desenvolvimento Front-End. Com a temática voltada para o lendário álbum de estreia \*\*"Ten"\*\* da banda \*\*Pearl Jam\*\*, o site une a paixão pela música com o aprendizado prático de estruturação, estilização e manipulação dinâmica de DOM.



\## 📝 Sobre o Projeto



O objetivo principal deste repositório é estritamente educacional. Ele serve como um ambiente de testes para aplicar conceitos de HTML semântico, CSS avançado (responsividade, flexbox, estilização de tabelas e menus) e JavaScript básico para automação de tarefas repetitivas (como a geração do menu de navegação).



\## 🚀 Funcionalidades e Estrutura



\* \*\*🏠 Página Inicial (`index.html` e `style1.css`):\*\* Apresenta um videoclipe da banda, uma tabela responsiva com informações técnicas sobre o álbum e um artigo formatado detalhando a história de "Ten".

\* \*\*🎵 Player de Música (`faixa1.html` e `style.css`):\*\* Estrutura de página dedicada à reprodução das faixas do álbum utilizando a tag `<audio>`. Possui botões estilizados para navegação entre as músicas.

\* \*\*🍔 Menu Dinâmico (`menu.js`):\*\* Um menu lateral estilo \*hamburger\* construído dinamicamente via JavaScript. O script contém um \*array\* com os nomes das faixas do álbum e injeta automaticamente os links no HTML, facilitando a manutenção sem precisar editar arquivo por arquivo.

\* \*\*ℹ️ Página "Sobre" (`about.html`):\*\* Uma seção dedicada a explicar o propósito do projeto, destacando a jornada de desenvolvimento Fullstack e esclarecendo o uso justo (\*fair use\*) dos materiais com direitos autorais.

\* \*\*⚠️ Página de Erro (`error.html`):\*\* Uma tela customizada simulando um erro de conexão (fallback visual) com um design amigável.



\## 💻 Tecnologias Utilizadas



\* \*\*HTML5:\*\* Estrutura semântica, integração de multimídia (`<audio>` e `<video>`) e atributos de acessibilidade (`aria-label`, `tabindex`, `role`).

\* \*\*CSS3:\*\* Variáveis, transições suaves, efeitos de \*backdrop-filter\* (desfoque no menu), pseudo-elementos (`::after`, `::before`) e \*Media Queries\* para garantir que o site funcione perfeitamente em dispositivos móveis.

\* \*\*JavaScript (ES6):\*\* Manipulação de arrays, uso de \*Template Literals\* e injeção de código via `innerHTML` para componentização do menu.



\## 🛠️ Como visualizar o projeto



Este é um projeto estático tradicional, o que significa que não requer configurações complexas ou servidores back-end para rodar:



1\. Faça o clone deste repositório na sua máquina.

2\. Certifique-se de que a estrutura de pastas contém as imagens (em `assets/img/`), vídeos (em `assets/video/`) e músicas (em `assets/music/`) caso queira a experiência multimídia completa.

3\. Abra o arquivo `index.html` em qualquer navegador web moderno.
