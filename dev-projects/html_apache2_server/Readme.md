# 🚀 DevSecOps Pipeline Simulator

Simulador interativo de pipeline **CI/CD com práticas de DevSecOps**,
desenvolvido com HTML, CSS e JavaScript puro.

## 📌 Visão Geral

Pipeline com etapas: - Checkout - SAST - Build - DAST - Deploy

Inclui: - Logs em tempo real - Métricas de segurança - Status dinâmico

## ⚙️ Tecnologias

-   HTML5
-   CSS3
-   JavaScript
-   Apache2

## 🚀 Como Executar

``` bash
git clone https://github.com/seu-usuario/devsecops-pipeline-simulator.git
cd devsecops-pipeline-simulator
```

Abrir:

``` bash
index.html
```

## 🌐 Deploy Apache2

``` bash
sudo apt install apache2
sudo cp index.html /var/www/html/
sudo systemctl restart apache2
```

Acessar: http://localhost
