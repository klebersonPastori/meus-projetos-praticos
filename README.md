### 🛡️ Active Directory Lab na AWS
> **Laboratório corporativo de AD em nuvem para estudos de segurança**

Provisionamento de instância **EC2 (Windows Server)** como Domain Controller, configuração de DNS interno, Organizational Units, usuários e GPOs. Voltado para estudo de superfície de ataque em ambientes Active Directory.

`AWS EC2` `Windows Server 2019` `Active Directory` `DNS` `GPO` `IAM`

---

### 🔍 Cyber Malicious File Risk Analyzer
> **Scanner de arquivos potencialmente maliciosos via CLI**

Scanner em **Python** para varredura recursiva de diretórios locais, identificando extensões de alto risco (`.exe`, `.bat`, `.vbs`, `.ps1`, `.dll`). Output estruturado com nível de risco por arquivo — apoio a análises forenses e triagem de incidentes.

`Python` `CLI` `os / pathlib` `Threat Detection` `Forensics`

---

### 🖥️ Network Troubleshooting Tools GUI
> **Interface gráfica para diagnóstico de redes corporativas**

Aplicativo desktop em **Python + PySide6** com execução **assíncrona** dos principais comandos de rede: Ping, Tracert, Netstat, NSLookup. Output em tempo real sem travar a UI — desenvolvido para acelerar troubleshooting no dia a dia corporativo.

`Python` `PySide6` `asyncio` `subprocess` `Networking`

---

### 📋 Incident Report — NIST CSF (DDoS/ICMP Flood)
> **Análise e resposta a incidente seguindo o NIST Cybersecurity Framework**

Documentação completa de resposta a ataque **DDoS via ICMP Flood**, seguindo as 5 funções do NIST CSF: Identify → Protect → Detect → Respond → Recover. Formato profissional de report técnico para SOC e liderança.

`NIST CSF` `DDoS` `ICMP Flood` `Incident Response` `Blue Team` `SOC`
